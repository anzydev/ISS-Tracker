Problem Statement
You need to build a complete web application (dashboard) that does 3 main things:

Tracks the International Space Station (ISS) live
Shows latest news articles
Includes an AI chatbot that answers questions based ONLY on your dashboard data
Think of it as a mini dashboard with live tracking, news, and a chatbot.



VIDEO Demonstration of APP: VIDEO (CLICK HERE)

PART 1: ISS (International Space Station) Live Tracking
You need to create a section that shows where the ISS is right now. 

What your app should do:
Get ISS location every 15 seconds automatically
Show it on an interactive map (use Leaflet.js)
Calculate and display speed (km/h)
Show last 15 positions as a path (trajectory)
Show nearest place (city/ocean name)
What should be visible on screen:
Latitude and Longitude
Speed of ISS
Current location name
Number of positions tracked
Line showing ISS path on map
Extra features:
Button to refresh manually
People in Space Right Now
Fetch the number of astronauts currently in space
Display:
Total number of people
List of names (if available)
API to Use:
ISS BASE API: http://api.openCopy Icon-notify.org/

**NOTE**

PART 6: Deployment

Requirements:
Deploy your app using Vercel
Share a live link in Google Form that anyone can access
Run the following commands in the Antigravity terminal:

1. To Install Vercel CLI
    npm i -g vercel

2. To log in to Vercel via CLI, use this command:
Vercel login
Optional (direct email login):
vercel login your-email@example.com

3. For First-time deployment
    vercel
(Select Yes → choose account → No for existing project → keep defaults)

4. To Deploy to production
    vercel --prod

5. For Future deployments
    vercel --prod
Deployment Steps:
Push your code to GitHub (public repository)
Go to Vercel.com and sign up
Connect your GitHub repository
Add environment variables in Vercel dashboard:
VITE_NEWS_API_KEY = Your NewsAPI key
VITE_AI_TOKEN = Your Hugging Face
Click Deploy
Copy the live URL
 IMPORTANT:

NEVER hardcode API keys in your code
Add .env to .gitignore
Configure environment variables in Vercel dashboard
Environment Variables Setup

Create .env file:

VITE_NEWS_API_KEY=your_newsapi_key_here

VITE_AI_TOKEN=your_huggingface_KEY
